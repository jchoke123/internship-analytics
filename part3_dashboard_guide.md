## What It Shows

| Visual | Insight |
|---|---|
| KPI Scorecards | Total applications, response rate, interviews, avg response days |
| Application Funnel | How applications progress from Applied → Interview → Rejected |
| Industry Bar Chart | Which industries have the best response rates |
| Monthly Timeline | Application activity and response rates over time |
| Resume Effectiveness | Tailored vs generic resume response rate comparison |

---

## How It Was Built

**Data source:** `internship_applications.csv` generated by `part1_data_tracking.py`

**Connection:** CSV uploaded to Google Sheets → connected to Looker Studio as a data source

**Key design decisions:**
- KPI cards placed at the top for immediate visibility
- Charts sorted by `status_rank` and `month_num` (not alphabetically) for correct ordering
- Filter bar added for industry — makes the dashboard interactive
- Colour coding: green = positive outcomes, red = rejections

---

## How to Rebuild It

1. Run `part1_data_tracking.py` to generate `internship_applications.csv`
2. Upload the CSV to Google Sheets
3. Go to [lookerstudio.google.com](https://lookerstudio.google.com)
4. Create new report → connect Google Sheets as data source
5. Add charts following the layout in `screenshots/dashboard_preview.png`
